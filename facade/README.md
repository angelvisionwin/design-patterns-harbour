## Facade ( fachada )

Proporcionar una interfaz unificada a un conjunto de interfaces en un subsistema. Fachada
define una interfaz de nivel superior que facilita el uso del subsistema.
El patrón tiene un propósito estructural y se aplica a los objetos.

### Cuándo usar

* desea proporcionar una interfaz simple a un subsistema complejo
* hay muchas dependencias entre los clientes y las clases de implementación de una abstracción
* desea superponer sus subsistemas, use una fachada para definir un punto de entrada a cada nivel de subsistema
